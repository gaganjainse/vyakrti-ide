# Vyākṛti Workbench

A browser-based IDE for Vyākṛti, an experimental Sanskrit-oriented programming language.

## Features
- Monaco-powered code editor with custom syntax highlighting
- File management (create, rename, delete .vya files)
- Compile & run code via Rust backend
- WebSocket REPL integration
- Dark/light theme
- Keyboard shortcuts
- AST, Tokens, and Bytecode viewer panels
- Diagnostics console

## Quick Start
npm install
npm run build
npm run dev
